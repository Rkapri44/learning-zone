HTML List

There are three different types of list in HTML

Ordered List

Ordered list are used to create a list of items in a specific order or sequence

  1. List item 1
  2. List item 2
  3. List item 3

Unordered List

Unordered list are used to create a list of items that are not in any order

  • Pears
  • Cheese
  • Apple

Definition List

Definition list are used to create a list of terms and their corrsponding definitions

Algorithm
An algorithm is a set of instructions or a step-by-step procedure for solving a problem or completing a task.

Nest List

You can also have a combonation of Ordered and Unordered Lists
