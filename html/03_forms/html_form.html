HTML Form

An HTML form is a section of a web page that allows users to input and submit data. It typically consists of various form elements like input fields, checkboxes, radio buttons, dropdown menus, and buttons.

Form Element - <form></form>

  • The form element is used to defeine a section of input elements that interact with the webpage.
  • The purpose of this element is to create a container for those input elements and to specifiy which data should be processed and submitted to the server

Fieldset Element - <fieldset></fieldset>

  • The fieldset is used to group related elements usually forms together.

Legend Element - <legendt></legend>

  • The legend element is used to provide a caption for the fieldset
